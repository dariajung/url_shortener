url_shortener
=============

A very simple URL shortener using Python, Flask, and Redis.

To get started:

```virtualenv venv```

```source venv/bin/activate```

```pip install -r requirements.txt```

```<path:to:redis> src/redis-server```

```python app.py```

Play around!
